# Keep Theme Starter

## Get start | 快速开始

### Configuring theme | 配置主题

Please configure `source/_data/keep.yml`

请在 `source/_data/keep.yml` 进行配置

### Install dependencies | 安装依赖

```bash
npm install
```

### Run the project | 运行项目

```bash
npm run server
```
